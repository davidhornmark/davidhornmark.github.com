Mobile Application - Selfnote

PA1469, Blekinge Institue of Technology, Karlskrona, Blekinge, Sweden
December, 2019
Project Information

As part of a Mobile Development Course, me and my team developed a mobile reminders application. We wanted to experiment a little and seize the opportunity to expand our reaches, that's why we chose to develop the application using the Flutter framework and Dart programming language. The choice was due to none of us having prior experience with it.

Idea

The application is targeting active mobile users in need of reminders in their daily life.

A reminder is only as good as its ability to help you remember, that’s why Selfnote's reminders always will be shown on the lock screen. You cannot miss it, and even if you would miss it Selfnote will notify you when the set time and place is reached.

Project Information

Small Team Project - NKT Cavedo

PA1416, Blekinge Institue of Technology, Karlskrona, Blekinge, Sweden
May, 2018
Project Information

The goal of the Cavedo project was to create a deeper Proof of Concept of a digital archive for NKT's testing protocols used in the testing of their high voltage cables. Previously the user had to manually fill in a paper protocol, later an Excel spreadsheet for internal storage of the data, and finally a Word document which served as a test protocol that the customer received.

By storing the test results in a database and automatically extract them when needed, Cavedo provided a streamlined way of collecting data, creating protocols for the customer and the internal system. This approach lowered the number of times a user had to input the same data, from three times to one and thus minimizing the risk of human error.

Cavedo was built as an internal website located at a local server which had the advantage of only needing one installation. By using responsive design, we allowed for the system to be used across a vast variety of devices as NKT had the idea of providing tablets for the employees. This made it so that wherever the employee was they would have access to all the reports and data and never have to print it out once, this being efficient and environment-friendly.

Individual Project - Christian Lundmark

PA1414, Blekinge Institue of Technology, Karlskrona, Blekinge, Sweden
October, 2017
Project Information

By using environment sensors, the gardening system will actively take decisions based on variables such as moisture, temperature, lighting, and nutrition in the soil.

Connecting these variables to events the greenhouse is one step closer to automation. Measuring the value of the variables allows for dynamic control over watering, blinders, heating, fertilizing, and checking whether windows and doors are open or closed.

The system provides easy access to administrative tools in used for managing sensors and activating functions manually.
